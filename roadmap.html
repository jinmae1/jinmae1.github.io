git

shell에서도 조금 사용하긴 하지만 VS Code의 익스텐션을 활용할 때가 많습니다. 기본적인 기능밖에 사용하고 있진 않지만, 빨리 숙달하고 복잡한 기능도 사용하고 싶습니다.

Maria DB

Oracle을 수업시간에 공부하긴 했지만 개인목적으로 가벼운 Maria DB를 선호합니다. 지난 추석 때 약간은 공부했지만 깊이 공부해야 합니다.

Mongo DB

github를 돌아다니다보면 mongodb를 사용한 프로젝트도 자주 보이기도 하고, 기본적인 기능은 관계형db보다 어려운 것 같지 않아 개인 목적으로 배워두고 싶습니다.

Python

학교에서 가르쳐준 C(++), Java 외에 처음으로 자력으로 공부해본 언어인데 C에 비해 현격하게 학습 난이도가 쉬워서 충격을 받았습니다. 가벼운 일괄처리 스크립팅이나 알고리즘 공부에 유용하게 쓰고 있습니다. 또한 github 상의 여러 프로젝트 등 python으로 이루어져 있어서 필요성이 굉장히 높습니다. 사칙연산처럼 익숙해지고 싶은 언어입니다.

nodejs

웹 프로젝트를 할 때 간단히만 써봤는데 유용한 라이브러리와 프로젝트들이 많아 잘 써먹었었습니다. 기존에 클로저와 콜백 함수 때문에 javascript를 포기했었는데 다시금 javascript를 배웠으니 한층 더 잘 활용할 수 있을 것 같습니다. 더불어 TypeScript도 미리 배워둬야 좋을 것 같습니다.

React

React가 핫한 만큼 얼마나 강력한지 한번 써보고 싶고 나아가 React Native도 손 대보고 싶습니다.

C#

개인적으로는 Java보다는 C#을 배우고 싶은데 나중에 여유가 있다면 C#을 공부하며 Java와 어떤점이 유사하고 어떤점이 다른지 비교해가며 공부해보고 싶습니다.

Kotlin

지난 번 Java 프로젝트를 하며, 그리고 js를 공부하며 null pointer 관련 문제가 얼마나 골때리는지 깨달았습니다. kotlin과 typeScript를 배우면서 null safe한 코드를 작성하는 법을 배우고 싶습니다.

Docker

리눅스를 다루다보면 항상 절망하는 부분 중 하나가 실행환경과 설정입니다. 설치 오류로 중도 포기하는 경우가 다반사인데 Docker가 아직 정확히 어떻게 돌아가는지 이해는 안 가지만 학습해두어 설정 지옥에서 벗어나고 싶습니다.
